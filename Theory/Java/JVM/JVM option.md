

### 분류
1. -X 옵션
* 비표준, 모든 JVM에서 지원하지는 않는다, 갑자기 해당 내용이 변경될 수 있다.
2. -XX 옵션
* 사용하는 것을 권장 X

### 종류
1. Java Heap Size
-Xms : 초기 heap size
-Xmx : 최대 heap size
2. GC log
3. Garbage Collector
4. Classpath
-Xbootclasspath/a : 부트스트랩 클래스의 앞에 위치하도록 설정
-Xbootclasspath/p : 부트스트랩 클래스의 뒤에 위치하도록 설정
